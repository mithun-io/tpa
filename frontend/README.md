# TPA Frontend - React + Vite Setup

## Requirements
Node.js (v18+)

## Setup Instructions

1. Ensure your Spring Boot backend is running on `http://localhost:8080` (with CORS enabled).
2. Open terminal in this `frontend` directory.
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
5. Open the provided localhost URL (usually `http://localhost:3000` or `http://localhost:5173`) in your browser.

## Tech Stack
- React.js + Vite
- React Router DOM for routing
- Tailwind CSS for styling
- Axios for API requests
- Lucide React for icons
- React Hot Toast for notifications

## Note
Do not run `npx create-vite` again. All files are already configured and created in this directory. Just run `npm install` and `npm run dev`.
